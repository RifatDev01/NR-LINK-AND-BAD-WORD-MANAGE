Telegram Anti-Link Bot

এই বটটি আপনার গ্রুপে কোনো মেম্বার লিংক পাঠালে তা সাথে সাথে ডিলিট করে দেবে।
কিভাবে সেটআপ করবেন:

১. বট টোকেন বসান:

    .env ফাইলটি ওপেন করুন।
    YOUR_BOT_TOKEN_HERE এর জায়গায় আপনার বট টোকেনটি বসান।

২. বট রান করুন:

    টার্মিনালে এই কমান্ডটি দিন:

    node index.js

৩. গ্রুপে অ্যাড করুন:

    বটটিকে আপনার টেলিগ্রাম গ্রুপে অ্যাড করুন।
    বটটিকে Admin হিসেবে সেট করুন এবং Delete Messages পারমিশন দিন।

বৈশিষ্ট্য:

    গ্রুপে লিংক (URL/Text Link) পাঠালে অটোমেটিক ডিলিট হবে।
    অ্যাডমিনদের জন্য এই নিয়ম কার্যকর হবে না (তারা লিংক পাঠাতে পারবেন)।
    লিংক ডিলিট করার পর একটি ওয়ার্নিং মেসেজ আসবে এবং ৫ সেকেন্ড পর সেটিও ডিলিট হয়ে যাবে যাতে গ্রুপ পরিষ্কার থাকে।
